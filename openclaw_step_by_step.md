## Step 23: Pre-Configure the Mac Mini at Home
- Before going to the client's location, set up as much as possible at home:
- **macOS setup:**
  - Turn on the Mac Mini, go through initial macOS setup
  - Create an admin account (for you to manage remotely)
  - Create a standard user account (for the client if they ever need to see the dashboard)
  - Update macOS to the latest version
- **Security setup:**
  - Enable FileVault (full disk encryption) — System Settings → Privacy & Security → FileVault → Turn On
  - Enable the firewall — System Settings → Network → Firewall → Turn On
  - Set a strong admin password (save in your password manager)
- **Remote access setup:**
  - Enable Remote Login (SSH):
    - System Settings → General → Sharing → Remote Login → Turn On
    - Allow access for your admin account only
  - Generate SSH key pair on your computer (if you haven't already)
  - Add your public key to the Mac Mini's authorized_keys
  - Test SSH connection from your computer to the Mac Mini (while on the same network at home)
- **Configure for 24/7 operation:**
  - System Settings → Energy → prevent sleep when display is off
  - System Settings → Energy → Start up automatically after a power failure
  - Enable auto-login for the admin account (so it starts up without needing a password on boot)
- **Install Claude:**
  - Sign into Claude with the client's account
  - Set up Claude to run on startup
- **Deploy the client's agents:**
  - Configure each agent with the prompts tailored to this client's business
  - Test every agent — make sure they respond correctly
  - Set up agents to auto-start on boot
- **Deploy the dashboard:**
  - Install the dashboard web app
  - Configure it with this client's data
  - Test it — make sure it loads and displays correctly
- **Set up backup script:**
  - Configure automated daily backup of all configs and data
  - Set backup destination (your VPS or cloud storage)
  - Test the backup manually — make sure it works
